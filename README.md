## Overview
Diet4Skin is a **Java Servlet-based web application** that helps users get a personalized 7-day diet plan based on their skin concerns.  
By selecting one or more skin problems, users receive daily food and drink suggestions aimed at improving their skin health.  
The app uses a **MySQL database** to store information about skin issues and healthy foods, and connects to it with **JDBC** for smooth data management.  
The clean **HTML/CSS frontend** provides a user-friendly interface for interacting with the app.

---

## Features

- **Symptom-Based Diet Planner**: Select one or more skin concerns (such as acne, dryness, or sensitivity) to instantly receive a custom diet plan focused on your skin health.  
- **7-Day Plan Generator**: Automatically generates a full week‚Äôs plan, with daily food and beverage recommendations tailored to your selected symptoms.  
- **MySQL Database Integration**: Stores skin issues, food categories, and diet plans in a structured MySQL database.  
- **JDBC + Servlets**: Backend logic is handled by Java Servlets, which interact with MySQL using JDBC for efficient data processing.  
- **HTML/CSS Frontend**: A responsive and simple web interface (`index.html`, `style.css`) makes it easy for users to input symptoms and view diet plans.  
- **Error Handling**: Handles incorrect inputs or missing data gracefully with informative messages.
